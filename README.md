# Structure

This repository contains the following directories:

- `sim_data/` – Simulation data file (histograms)  
- `scripts/` – Plotting scripts  
- `results/` – Output figures  

## Generating the DVpi0P Figure for preTDR

To generate the DVpi0P figure, run the following commands:

```bash
cd scripts
root -l plots_DVpi0P.C

The generated figure will be saved in the `results/` directory.
